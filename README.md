# Simple RAG Application

A Retrieval-Augmented Generation (RAG) application built with Streamlit, ChromaDB, and Google's Gemini API.

## Getting Started

This repository contains a complete RAG application. Please see the development branch for the full implementation.

## Quick Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up your environment variables
4. Run the application: `streamlit run app.py`
